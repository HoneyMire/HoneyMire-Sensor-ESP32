🍯HoneyOpus

An ESP32 SSH/Telnet honeypot — installable from your browser.

Targets: ESP32-C3 · ESP32-S3 Build: latest Built: —

Install the latest firmware

Plug your board in via USB (ESP32-C3 SuperMini, LilyGO T-QT Pro or ESP32-S3 N16R8), then click Connect and pick the serial port. The flasher auto-detects the chip family and picks the matching image.

Your browser can't talk to USB serial. Use a recent Chrome, Edge or Opera on a desktop computer.

What is this?

HoneyOpus is a pocket-sized honeypot that fakes a sloppy Ubuntu server on ports 22 and 23. It records every attacker session as an asciinema cast, classifies the attack profile (Mirai, IoT loader, manual operator, …), geolocates the source, and can optionally report to AbuseIPDB and AlienVault OTX.

After flashing

  1. The display (if your board has one) briefly shows the boot logo, then turns off.
  2. The board comes up as a SoftAP HoneyOpus-XXXXXX (password honeyopus). Connect with your phone — the captive portal opens automatically.
  3. Pick your Wi-Fi network, enter the password, hit Connect. The device reboots and joins your LAN.
  4. Browse to the IP shown on the display (or in the serial console for headless boards). Default dashboard login is admin / honeyopus (LAN clients are auto-allowed without auth — change the password before exposing the device).

Using it safely

Run it on a network segment you don't care about. Forward only TCP 22 and 23 from your router. The fake shell never executes anything — it returns canned strings — but no honeypot is hardened against a determined adversary. Never leave the default dashboard password if the device is reachable from the internet.
